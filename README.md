# FUTURE_DS_03 
Marketing Funnel & Conversion Performance Analysis

This repository contains a Power BI dashboard designed to analyze marketing funnel performance and evaluate conversion efficiency across different stages and channels.

Tools Used

Power BI Desktop

CSV / Excel data source

Key Analysis

Funnel stages: Impressions, Clicks, Leads, and Conversions

Conversion rate and drop-off analysis

Channel-wise marketing performance

Trend analysis of campaign performance

Files in This Repository

Marketing_Funnel_Conversion.pbix – Power BI dashboard file

dashboard_preview.png – Dashboard screenshot

README.md – Project documentation

How to View

Download the .pbix file and open it using Power BI Desktop.
